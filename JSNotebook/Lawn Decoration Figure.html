My JavaScript Notebook: Graphics Objects

What I learned today is how to add shapes into documents. I learned how to prepare the canvas and write code so that the computer will draw out the correct graphic. The two shapes that were explored in this chapter were the rectangle and the circle. For both images a variable is setup with the shapes cordinates. The rectangle has the x and y points and a height and width that need to be set while the circle has a radius value instead of height and width. After setting the shape up you can set the color of the shape. Afterwards you have to add the image to the canvas and then tell the it to draw the shape. You can use math commands to make the shapes certain sizes and proportions and make the width and height the same sizes as the canvas. This module talked about if statements. These statements as a user about a statement and if that statement is correct then the computer will do some function. IF that function is not correct then the computer will do something else. This funtion is known as the if else fuction. These tell the computer if some condition is met then do this action, but if the condition is not met then do this other action.
